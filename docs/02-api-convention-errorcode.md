# 接口规范与全局错误码（前后端 + Codex 对齐基线）

> 所有接口遵循此规范，任何字段/格式/错误码变更须评审后统一更新，禁止私自定义。

---

## 1. 通用约定

- Base URL：`/api`
- 编码：UTF-8；请求/响应体：`application/json`（文件上传除外）。
- 时间：ISO 8601 带时区，如 `2026-07-04T10:00:00+08:00`。
- 鉴权：请求头 `Authorization: Bearer <accessToken>`（外部签署接口除外）。
- 分页：入参 `page`(默认1) / `pageSize`(默认20, 上限100)；出参 `list` + `total` + `page` + `pageSize`。

## 2. 统一响应格式

成功：
```json
{ "code": 0, "message": "success", "data": { }, "traceId": "..." }
```
分页：
```json
{ "code": 0, "message": "success",
  "data": { "list": [], "total": 100, "page": 1, "pageSize": 20 },
  "traceId": "..." }
```
失败：
```json
{ "code": 40101, "message": "登录已失效，请重新登录", "data": null, "traceId": "..." }
```
> `code=0` 表示成功；非 0 即业务错误码。HTTP 状态码同时使用（401/403/400/500 等），前端以 `code` 字段为准做业务判断。`traceId` 用于日志排障。

## 3. 文件上传约定
- 方式：`multipart/form-data`。
- 合同：仅 `application/pdf`，magic number 校验 `%PDF`，大小 ≤ 50MB。
- 印章：仅透明 `image/png`，大小 ≤ 5MB。
- 个人签名/印章（签署端）：png/jpeg，≤ 5MB。
- 服务端计算 MD5 去重、SHA256 留存；返回存储引用（不返回真实路径）。

## 4. 请求头
| Header | 必填 | 说明 |
|---|---|---|
| Authorization | 内部接口必填 | `Bearer <token>` |
| X-Request-Id | 可选 | 客户端可传，用于链路追踪，缺省服务端生成 |
| Content-Type | 是 | json 或 multipart/form-data |

## 5. 全局错误码表

> 结构：`AABCC` — AA 模块，B 类别，CC 序号。前端按 code 提示，后端 `ErrorCode` 枚举统一维护。

### 通用 (10xxx)
| code | 说明 |
|---|---|
| 0 | 成功 |
| 10001 | 参数校验失败 |
| 10002 | 请求体格式错误 |
| 10003 | 资源不存在 |
| 10004 | 请求过于频繁（限流） |
| 10005 | 系统内部错误 |
| 10006 | 操作不被允许（状态机非法流转） |

### 认证鉴权 (40xxx)
| code | 说明 |
|---|---|
| 40101 | 未登录 / token 无效或过期 |
| 40102 | 账号或密码错误 |
| 40103 | 账号已禁用 |
| 40104 | 账号已锁定（登录失败过多） |
| 40105 | refresh token 无效 |
| 40301 | 无权限访问（RBAC 拦截） |
| 40302 | 无该操作按钮权限 |

### 用户/角色 (41xxx)
| code | 说明 |
|---|---|
| 41001 | 用户名已存在 |
| 41002 | 角色不存在 |
| 41003 | 不可删除/禁用超级管理员 |

### 印章 (42xxx)
| code | 说明 |
|---|---|
| 42001 | 印章文件格式不合法（须透明PNG） |
| 42002 | 印章文件超过大小限制 |
| 42003 | 印章已存在（MD5 重复） |
| 42004 | 印章已禁用，不可使用 |
| 42005 | 无权预览/下载印章原图 |

### 合同 (43xxx)
| code | 说明 |
|---|---|
| 43001 | 仅支持 PDF 文件 |
| 43002 | 文件超过大小限制 |
| 43003 | 合同编号已存在 |
| 43004 | 文件重复（MD5 已存在） |
| 43005 | 文件安全检测未通过 |
| 43006 | 合同状态不允许该操作 |

### 签署任务 (44xxx)
| code | 说明 |
|---|---|
| 44001 | 签署任务不存在 |
| 44002 | 任务状态不允许该操作 |
| 44003 | 印章坐标参数非法 |
| 44004 | 截止时间非法（须晚于当前时间） |
| 44005 | 任务已签署，不可撤回 |

### 签署端 (45xxx)
| code | 说明 |
|---|---|
| 45001 | 签署链接无效 |
| 45002 | 签署链接已过期 |
| 45003 | 签署链接已使用 |
| 45004 | 签署任务已作废 |
| 45005 | 签名/印章文件不合法 |
| 45006 | 签署人身份校验失败 |

### 归档审计 (46xxx)
| code | 说明 |
|---|---|
| 46001 | 无权下载归档文件 |
| 46002 | 文件哈希校验不通过（文件可能被篡改） |
| 46003 | 上传的校验文件格式错误 |
