# 内部电子签章系统 · 后端

NestJS + PostgreSQL + Redis + MinIO 的企业内部自用电子签章平台后端。

## 业务流程
上传 PDF 合同 → 管理员盖企业印章 → 创建签署任务生成一次性链接 → 对方在线签名/盖章回传 → 生成防篡改归档件 → 任务创建人收到站内提醒。
